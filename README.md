# OSA

**Один запрос — чистая правда за 2 секунды.**

Пишешь что угодно → получаешь чёткую сводку без воды, с источниками и уровнем достоверности.

Всё в браузере. Ничего не ставишь.

## Что умеет
- Молниеносно ищет по Wikipedia
- Выкидывает повторы и противоречия
- Показывает откуда взял и насколько можно верить
- 2 KB кода • 0 серверов • 0 ключей

## Запуск
1. Клонируем репозиторий: `git clone https://github.com/feedod/osa.git`
2. Открываем в браузере: `open index.html`
